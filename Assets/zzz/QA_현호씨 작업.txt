#편의성 관련 의견
- ! 버튼으로 상호작용이 raycast forward로 활성화 되는것같은데 방향 무관하게 가까이서 활성화되면 더 편할 것 같아요
- 필드에 카드들을 다시 확인하려고 닫고 싶을 때 카드 제시창을 못닫아서 답답해요
- 필드 좌측, 정면, 우측에 누가 있는건지 얼굴 이미지나 이름을 표시해주면 더 편할 것 같아요
(딜러의 텍스트를 봐도 게임 도우미로 다시 확인해야됨)
- 진행하다 규칙이 헷갈려서 ?버튼으로 규칙을 확인할 때 게임은 진행되고 있어서 딜러의 대사창이 규칙 창을 가려요
- 게임 도우미 대상 선택을 한 대상이 체크돼있는 상태에서도 바로 다른 대상을 체크하면 바뀌도록 하면 편할 것 같아요
(지금은 체크 풀고 다른 대상 체크)
- UI 선택했을 때 같은 대사가 계속 나오는게 스킵버튼이 있더라도 너무 불편해요
